## Instalation

```
yarn
yarn dev
```

```
truffle compile

```

### Deploy contract
```
truffle migrate
```

## Troubleshooting
If you have problems with truffle migrate, try this:
change "type": "commonjs" instead of "type": "module" in package.json,

### Dependencies
* @types/web3
* web3
* truffle
